1. download and start redis
docker pull redis 
docker run -p 6379:6379 --name redis_instance -td redis

2. download and start phore db updater
clone this repo:
git clone https://github.com/phoreproject/Phore-RPC-Proxy
cd ./script/keepPhoreUpdated
docker build -t keep_phore_updated .
docker run -t keep_phore_updated

for sharing docker volume with host add additional docker run parameter:
-v LOCAL_BACKUP_DIR:/data/backup_dir

3. Set environmental variables
REDIS_HOST = redis docker ip
REDIS_PORT = 6379 (default)

4. Start phored instance
to connect to previously runned redis instance you need to connect containers to same network
- docker network create phoreNet
- set correct REDIS_HOST and REDIS_PORT in phored/Dockerfile
- cd ./script/phored
- docker build -t phored .
- docker run -td phored

and connect all runned containers to net:
docker network connect phoreNet $(docker ps -q)
